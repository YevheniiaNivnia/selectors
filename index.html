Селектори

  • CSS Селектор елемента

    Селектор елементів обирає HTML елементи на основі імені елемента. Використовується назва елемента
    p {text-align: center; color: red;} - всі елементи < p > будуть розміщені по центру зі шрифтом червоного коляру

  • CSS Селектор id

    Селектор id для вибору визначенного елемента використовує атрибут id HTML елемента. Використовується (#)
    #selector-id { font-size: 16px; color: blue; } - елемент з id="selector-id" - буде зі шрифтом розміром 16px та синього коляру

  • CSS Селектор class

    Селектор class обирає HTML елементи з визначенним атрибутом класу. Використовується (.)
    .h3 { font-size: 25px; color: blueviolet; } - всі елементи параграфу h3 будуть розміром 25px зі шрифтом фіолетового коляру

  • CSS Селектор універсальний

    Універсальний селектор виділяє всі елементи Html на сторінці. Використовується (*)
    * { background-color: rgb(202, 203, 196); } - означає, що фон сторінки буде сірого коляру

  • CSS Селектори групування

    Селектор групування обирає всі HTML елементи з однаковими визначеннями стилю
    h1, h2, p { text-align: center; color: red; } - означає, що елементи заголовку h1, h2 та параграфу розміщені по центру зі шрифтом червоного коляру
